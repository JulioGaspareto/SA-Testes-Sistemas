# ENTREGA 01: Requisitos Funcionais

| ID    | Requisito           | Descrição                                                                                  |
| ----- | ------------------- | ------------------------------------------------------------------------------------------ |
| RF-01 | Cadastro de usuário | O sistema deve permitir que novos usuários criem uma conta informando nome, email e senha. |
| RF-02 | Login de usuário    | O sistema deve permitir que usuários cadastrados façam login utilizando email e senha.     |
| RF-03 | Logout              | O sistema deve permitir que o usuário encerre sua sessão.                                  |
| RF-04 | Visualizar perfil   | O usuário deve poder visualizar suas informações cadastradas.                              |
| RF-05 | Editar perfil       | O usuário deve poder editar suas informações pessoais.                                     |
| RF-06 | Criar tarefa        | O usuário deve poder cadastrar tarefas ou lembretes.                                       |
| RF-07 | Editar tarefa       | O usuário deve poder editar tarefas cadastradas.                                           |
| RF-08 | Excluir tarefa      | O usuário deve poder excluir tarefas cadastradas.                                          |
| RF-09 | Visualizar tarefas  | O sistema deve exibir a lista de tarefas cadastradas pelo usuário.                         |
| RF-10 | Pesquisa de tarefas | O sistema deve permitir pesquisar tarefas ou rotinas cadastradas.                          |




# ENTREGA 02: Requisitos Nao Funcionais
| ID     | Requisito       | Descrição                                                                                 |
| ------ | --------------- | ----------------------------------------------------------------------------------------- |
| RNF-01 | Usabilidade     | O sistema deve possuir interface simples e intuitiva para facilitar o uso pelos usuários. |
| RNF-02 | Desempenho      | O sistema deve carregar páginas em até 3 segundos.                                        |
| RNF-03 | Segurança       | As senhas dos usuários devem ser armazenadas de forma criptografada.                      |
| RNF-06 | Responsividade  | O sistema deve se adaptar a diferentes tamanhos de tela (celular, tablet e computador).   |


 # O sistema proposto consiste em um site de gerenciamento de tarefas, rotinas e lembretes, desenvolvido com o objetivo de auxiliar os usuários na organização de suas atividades do dia a dia. A plataforma permitirá que os usuários criem uma conta e realizem login para acessar suas informações pessoais. Após autenticados, os usuários poderão cadastrar, visualizar, editar e excluir tarefas, rotinas e lembretes, facilitando o planejamento e acompanhamento de suas atividades. O sistema terá como foco ajudar os usuários a não esquecer compromissos importantes, além de contribuir para uma melhor organização pessoal e produtividade no cotidiano. As tarefas cadastradas poderão ser consultadas facilmente dentro do sistema, permitindo que o usuário visualize suas atividades de forma clara e organizada. Além disso, o sistema buscará oferecer uma interface simples e intuitiva, possibilitando que qualquer usuário consiga utilizar a plataforma sem dificuldades, independentemente do nível de experiência com tecnologia. O site será acessado por meio de navegadores web e permitirá que cada usuário gerencie suas próprias tarefas de maneira segura e personalizada.
